# Experiment-Management-System

单目头部视频三维重建实验管理系统

---

## 📌 项目简介
本项目是一个轻量级科研实验管理系统，用于管理视频数据、实验任务及结果展示。

系统重点是：
- 管理实验流程
- 展示实验结果
- 支持实验对比与报告导出

---

## 🚀 功能模块（MVP）

- 登录模块
- 项目管理
- 视频管理
- 实验任务管理
- 实验结果展示
- 实验对比
- 报告导出

---

## 📂 项目结构

```text
frontend/       前端代码
