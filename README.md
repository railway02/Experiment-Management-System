Experiment Management System

单目头部视频三维重建实验管理系统

📌 项目简介

本项目是一个轻量级科研实验管理系统，面向单目头部视频三维重建任务，支持实验数据管理、流程组织与结果分析。

系统核心目标：

🔄 规范实验流程管理

📊 可视化展示实验结果

📈 支持实验对比分析

📄 一键导出实验报告

🚀 功能模块（MVP）

当前最小可用版本（MVP）包含以下模块：

🔐 登录模块

📁 项目管理

🎥 视频管理

🧪 实验任务管理

📊 实验结果展示

⚖️ 实验对比分析

📄 报告导出

📂 项目结构
frontend/        前端代码
backend/         后端代码
docs/            项目文档
data/            上传数据与实验结果
demo_results/    演示用结果
screenshots/     系统截图
scripts/         辅助脚本
🌿 分支规范

为保证协作清晰，采用以下分支策略：

main：稳定版本（可用于演示/发布）

dev：开发集成分支

feature/<姓名>-<模块名>：个人功能开发分支

示例：

feature/zhangsan-video-module
⚙️ 协作规范

📌 所有代码必须提交到对应目录，禁止随意堆放

📄 文档统一存放在 docs/ 目录

🚫 禁止通过聊天工具传递“最终版文件”

🔄 每日同步开发进度，保持信息透明
